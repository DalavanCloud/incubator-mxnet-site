Detailed Description

a std::istream class that can can wrap Stream objects, can use istream with that output to underlying Stream

Usage example:

Stream *fs = Stream::Create("hdfs:///test.txt", "r");
dmlc::istream is(fs);
is >> mydata;
delete fs;
